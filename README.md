# BuildBoard

BuildBoard is a full stack project management and collaboration platform built to practice modern software engineering.

## Planned Stack

- Frontend: React, TypeScript, Tailwind CSS
- Backend: Node.js, Express, TypeScript
- Database: PostgreSQL
- Dev Tools: Docker, GitHub Actions, Postman

## Planned Features

- User authentication
- Workspaces
- Projects
- Tasks
- Comments
- Roles and permissions
- Dashboard
- Deployment

## Project Goal

The goal of BuildBoard is to learn and build a production-style full stack application from scratch.
